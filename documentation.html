AutoRia Clone - Documentazione

Sistema completo di vendita auto con backend NestJS/TypeORM e frontend React

Panoramica del Progetto

AutoRia Clone è un'applicazione web completa per la vendita di automobili, ispirata alla piattaforma AutoRia. Il sistema è composto da un backend sviluppato con NestJS e TypeORM, e un frontend sviluppato con React.

L'applicazione supporta diverse tipologie di utenti (Acquirenti, Venditori, Manager, Amministratori) e tipi di account (Base e Premium), con funzionalità differenziate per ciascuno.

Architettura del Sistema

Backend (NestJS/TypeORM)

Il backend è sviluppato utilizzando NestJS, un framework per la creazione di applicazioni server-side efficienti e scalabili in Node.js, insieme a TypeORM per la gestione del database.

Caratteristiche principali:

  • Architettura modulare con separazione delle responsabilità
  • Sistema di ruoli e permessi flessibile (RBAC)
  • Autenticazione e autorizzazione con JWT
  • Validazione automatica degli annunci
  • Supporto multi-valuta con aggiornamento quotidiano dei tassi
  • Containerizzazione con Docker per facile deployment

Frontend (React)

Il frontend è sviluppato con React, utilizzando Material-UI per l'interfaccia utente, Redux per la gestione dello stato e React Router per la navigazione.

Caratteristiche principali:

  • Interfaccia utente moderna e responsive
  • Gestione centralizzata dello stato con Redux
  • Autenticazione e autorizzazione basata su token JWT
  • Ricerca avanzata degli annunci
  • Dashboard utente per la gestione degli annunci
  • Visualizzazione dettagliata degli annunci

Requisiti di Sistema

Per lo sviluppo e l'esecuzione locale:

  • Node.js (v16 o superiore)
  • npm o yarn
  • Docker e Docker Compose
  • PostgreSQL (se non si utilizza Docker)

Per il deployment:

  • Server con supporto per Docker
  • AWS o altro provider cloud (opzionale)

Installazione e Configurazione

Backend

# Clonare il repository
git clone [repository-url]
cd autoRiaClone

# Installare le dipendenze
npm install

# Configurare le variabili d'ambiente
cp .env.example .env
# Modificare il file .env con i propri valori

# Avviare con Docker
docker-compose up -d

# Oppure avviare in modalità sviluppo
npm run start:dev

Frontend

# Navigare nella directory del frontend
cd autoRiaClone-frontend

# Installare le dipendenze
npm install

# Avviare in modalità sviluppo
npm start

API Endpoints

Il backend espone una serie di API RESTful per l'interazione con il frontend. La documentazione completa delle API è disponibile tramite Swagger all'indirizzo http://localhost:3000/api quando l'applicazione è in esecuzione.

Principali endpoint:

Endpoint Metodo Descrizione
/auth/register POST Registrazione nuovo utente
/auth/login POST Login utente
/users GET Elenco utenti (richiede autenticazione)
/cars/brands GET Elenco marche auto
/cars/brands/:id/models GET Elenco modelli per marca
/ads GET Elenco annunci
/ads POST Creazione nuovo annuncio (richiede autenticazione)
/ads/:id GET Dettaglio annuncio

Funzionalità Principali

Sistema di Ruoli e Permessi

L'applicazione implementa un sistema di controllo degli accessi basato su ruoli (RBAC) con quattro ruoli principali:

  • Acquirente: Può visualizzare annunci e contattare venditori
  • Venditore: Può creare, modificare ed eliminare i propri annunci
  • Manager: Può gestire annunci e utenti
  • Amministratore: Ha accesso completo a tutte le funzionalità

Tipi di Account

Sono disponibili due tipi di account con funzionalità differenziate:

  • Base: Limitato a un solo annuncio attivo
  • Premium: Annunci illimitati e accesso a statistiche avanzate

Gestione Annunci

Gli utenti possono creare, modificare, eliminare e visualizzare annunci di auto. Gli annunci includono informazioni dettagliate sul veicolo, immagini e dati di contatto del venditore.

Ricerca Avanzata

Il sistema offre funzionalità di ricerca avanzata per trovare veicoli in base a marca, modello, prezzo, anno, chilometraggio e altre caratteristiche.

Supporto Multi-valuta

I prezzi possono essere visualizzati in diverse valute, con tassi di cambio aggiornati quotidianamente.

Guida all'Uso

Per gli Acquirenti

  1. Registrarsi o accedere con un account esistente
  2. Utilizzare la ricerca per trovare veicoli di interesse
  3. Visualizzare i dettagli degli annunci
  4. Contattare i venditori per maggiori informazioni

Per i Venditori

  1. Registrarsi o accedere con un account esistente
  2. Accedere alla dashboard
  3. Creare un nuovo annuncio con i dettagli del veicolo
  4. Gestire gli annunci esistenti (modifica, eliminazione)
  5. Rispondere alle richieste degli acquirenti
